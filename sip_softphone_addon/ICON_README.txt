# Icon

This addon needs an icon.png file (256x256 pixels).

You can create one or use a placeholder. For now, you can create a simple one with:

```bash
convert -size 256x256 xc:blue -fill white -pointsize 72 -gravity center -annotate +0+0 "SIP" icon.png
```

Or download a phone icon from https://materialdesignicons.com/
